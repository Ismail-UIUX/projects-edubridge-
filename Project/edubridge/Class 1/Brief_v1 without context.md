
Project: CRM Dashboard Experience Design
1. Original Project Brief (Without Context)

Design a mobile-first CRM platform for sales teams and customer support managers.

The platform should help users:

manage leads
track customer interactions
monitor sales pipelines
assign follow-up tasks
view customer activity history
manage support conversations
receive real-time notifications

The goal is to improve team productivity, reduce missed follow-ups, and simplify customer management workflows.

Target users:

Sales representatives
Customer support agents
Team managers

Key requirements:

Clean and easy-to-understand dashboard
Fast navigation between customer records
Simple task management flow
Clear customer activity timeline
Mobile usability considerations
Notification and reminder system
Lightweight and scalable UI system

Please provide:

UX strategy
user flow recommendations
dashboard structure
feature prioritization
mobile usability suggestions
onboarding recommendations
UI/UX improvement opportunities
